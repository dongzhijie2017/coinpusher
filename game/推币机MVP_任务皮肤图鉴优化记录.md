# 推币机MVP任务皮肤图鉴优化记录

## 实施策略

参考用户提供的单文件 `index.html` 方案，但不直接替换当前 Vite + TypeScript + Phaser 工程。

原因：

- 当前项目已接入 GitHub、Nginx、HTTPS 和部署脚本。
- 单文件 HTML 依赖 CDN 版 Phaser/Tailwind/Lucide，不利于长期构建和小游戏迁移。
- 当前工程更适合继续拆模块、做微信小游戏适配和 Netlify/服务器自动部署。

## 已实现内容

### Gem 与皮肤内循环

新增资源：

```text
Gem
```

用途：

- 完成每日任务后获得。
- 用于外观商店解锁硬币皮肤。

新增皮肤：

- 原版金币
- 钛钢银币
- 熔岩红晶
- 翡翠流光

皮肤支持热切换：

- 新生成硬币使用当前皮肤。
- 台面已有硬币会同步切换贴图。

### 每日任务

已实现 3 个任务：

- 今日投币 20 次。
- 命中中央槽 5 次。
- 单局收集 150 Gold。

任务达成后可领取 Gem。

### 修复图鉴

新增街机修复目标：

- 街头篮球机
- 像素迷宫机
- 太空防卫机
- 极速赛车机

Gold 满足当前目标后，可在图鉴面板中修复下一台街机。

### 离线收益

离线收益已接入：

- 最多计算 8 小时。
- 离线期间恢复 Coin。
- 离线期间按图鉴槽等级获得 Gold。

### 合规保持

继续避免以下表达：

- 抽奖
- 中奖
- 爆率
- 返现
- 提现
- 以小博大

保留表达：

- 机关检索
- 修复图鉴
- 零件扫描
- 外观商店
- 每日任务
