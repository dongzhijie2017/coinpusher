
```text
生成一张游戏美术资产图，用于推币机游戏硬币sprite切片。内容：金色游戏币、银色游戏币、蓝色特殊币各一枚，每枚硬币包含正面、轻微侧转、强高光、弱高光四种状态，整齐排列在透明或纯色背景上。风格：精致金属材质，移动游戏资产，边缘清晰，可裁切成64x64或128x128。不要文字、不要品牌logo、不要现金符号、不要真实货币图案。
```

处理建议：

- Lovart 原图建议 512x512 或更高。
- Web 端压缩成 64x64 或 128x128。
- 做 8-12 帧轻微翻转高光，投币时播放序列帧。
- 不使用真实货币图案，避免合规风险。
