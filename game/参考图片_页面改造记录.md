# 参考图片页面改造记录

日期：2026-06-21

## 参考取舍

本次参考 `参考图片.md` 中的三张图，只吸收信息层级和移动端交互结构，不复刻直播间、送礼、排行榜、概率表和付费购买语境。

- 主界面参考点：集中展示一台主机台、底部大操作按钮、右侧快捷入口、规则入口常驻。
- 补给页参考点：移动端底部抽屉、明确数量选项、主按钮决策清晰。
- 规则页参考点：独立说明页、按条目解释玩法，但去掉概率营销和礼物购买包装。

## 已落地改造

- 主视觉改为红金街机招牌、蓝色玻璃推币台、三通道结算槽。
- 主操作区改成「补给 / 点击推币 / 自动开关」三段式，降低底部拥挤度。
- 右侧入口改成「规则、升级、图鉴、任务、外观」，常用与成长功能分层。
- 升级从主界面移入「机器升级」弹层，减少常驻小字和遮挡。
- 新增自动投币存档字段，支持开关状态持久化。
- 调整 Matter 重力为浅坡面重力，金币会在台面堆叠并被推板推动，不再快速漏空。
- 开放底部物理边界，金币越过结算线后再按左右回收槽或中央修复槽发放 Gold。
- 补给上限拆分为自然恢复上限 50 和补给临时上限 80，避免补给后被自然恢复逻辑扣回。

## 规则与合规边界

- Coin 只用于投币，自然恢复或通过维护补给获得。
- Gold 来自掉落结算，用于升级机器和修复图鉴。
- Gem 来自每日任务，只用于硬币外观。
- 小黄鸭触发的是免费「机关检索」，不售卖检索次数。
- 不使用「抽奖、中奖、爆率、提现、返现、礼物购买」作为核心包装。
- 不提供现金、实物、提现、交易或返利奖励。
