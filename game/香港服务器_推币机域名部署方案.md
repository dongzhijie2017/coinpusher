# 香港服务器推币机域名部署方案

## 推荐域名

将原示例里的 `demo.xhscopy.top` 替换为推币机项目域名：

```text
coinpusher.xhscopy.top
```

最终访问地址：

```text
https://coinpusher.xhscopy.top
```

## 部署结构

服务器目录建议：

```text
/var/www/coinpusher/
  dist/
    index.html
    assets/
```

本地项目已提供：

```text
deploy/deploy_hk.sh
deploy/nginx/coinpusher.xhscopy.top.conf
```

## 本地构建与上传

在本地项目目录执行：

```bash
./deploy/deploy_hk.sh root@你的香港服务器IP /var/www/coinpusher
```

脚本会执行：

1. `npm run build`
2. 通过 SSH 创建远程目录
3. 用 `rsync` 同步 `dist/` 到服务器

## Nginx 配置

将配置复制到服务器：

```bash
scp deploy/nginx/coinpusher.xhscopy.top.conf root@你的香港服务器IP:/etc/nginx/sites-available/coinpusher.xhscopy.top.conf
```

启用站点：

```bash
ssh root@你的香港服务器IP
ln -s /etc/nginx/sites-available/coinpusher.xhscopy.top.conf /etc/nginx/sites-enabled/coinpusher.xhscopy.top.conf
nginx -t
systemctl reload nginx
```

如果服务器使用的是 `/etc/nginx/conf.d/` 结构，则放到：

```text
/etc/nginx/conf.d/coinpusher.xhscopy.top.conf
```

## 阿里云 DNS 配置

如果 `xhscopy.top` 在阿里云管理：

| 配置项 | 值 |
| --- | --- |
| 记录类型 | A |
| 主机记录 | coinpusher |
| 记录值 | 香港服务器公网 IP |
| TTL | 默认 10 分钟 |

如果前面还有 CDN 或反向代理，也可以使用 CNAME：

| 配置项 | 值 |
| --- | --- |
| 记录类型 | CNAME |
| 主机记录 | coinpusher |
| 记录值 | CDN/代理提供的域名 |
| TTL | 默认 10 分钟 |

## HTTPS 证书

推荐使用 Certbot 自动签 Let's Encrypt：

```bash
apt update
apt install -y certbot python3-certbot-nginx
certbot --nginx -d coinpusher.xhscopy.top
```

Certbot 成功后会自动改写 Nginx 配置并启用 HTTPS。

## 验证命令

DNS：

```bash
dig coinpusher.xhscopy.top
```

HTTP：

```bash
curl -I http://coinpusher.xhscopy.top
```

HTTPS：

```bash
curl -I https://coinpusher.xhscopy.top
```
